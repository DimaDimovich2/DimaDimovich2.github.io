Это заполнитель для директории images.

Для полной функциональности OTIScord требуются следующие изображения:
- logo.png - логотип OTIScord
- favicon.png - иконка сайта
- server1.png, server2.png - иконки серверов
- avatar.png, avatar1.png, avatar2.png, avatar3.png, avatar4.png - аватары пользователей

Вы можете добавить свои собственные изображения, заменив файлы с теми же именами.

Директория images содержит SVG-изображения для OTIScord.

В приложении используются следующие SVG-файлы:
- logo.svg - логотип OTIScord
- favicon.svg - иконка сайта
- server1.svg, server2.svg - иконки серверов
- avatar.svg, avatar1.svg, avatar2.svg, avatar3.svg, avatar4.svg - аватары пользователей

Все изображения выполнены в векторном формате SVG для обеспечения высокого качества при любом масштабе.
